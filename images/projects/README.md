# Public Project Photos

Only place approved web-ready images here.

Important:
- Everything under `images/` is publicly accessible after deploy.
- Do not put private originals here.
- Use this folder only for curated images that are good enough to represent the brand.

Folders:
- `in-progress/`: clean active-job photos that show real work in motion
- `completed/`: strongest finished spaces and final results

Best first placements on this site:
- `index.html`: one strong wide photo in the hero or a small credibility strip near the top
- `about.html`: active projects, walkthroughs, and recently completed work
- `residential.html`: hero replacement or a small project showcase section
- `bathroom-remodeling-philadelphia.html`: strongest finished bathroom first
- `kitchen-remodeling-philadelphia.html`: strongest finished kitchen first
- `full-gut-rehab-philadelphia.html`: in-progress demo, framing, or rough-stage photos
- `contact.html`: only a small proof strip if the image feels clean and trustworthy

Use finished photos for:
- hero sections
- service page headers
- before-and-after style proof
- high-trust sales sections

Use in-progress photos for:
- process sections
- active project sections
- walkthrough credibility
- full-gut rehab content

Recommended prep:
- prefer landscape images when possible
- export around 1600 to 2200 pixels wide
- use optimized JPG or WebP
- use descriptive file names like `south-philly-bathroom-finished-01.jpg`

Rule of thumb:
- A real but average photo is often better than a stock image in a supporting section.
- A weak real photo is usually worse than no photo in a hero section.
